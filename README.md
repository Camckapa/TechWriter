This is a technical writer's portfolio and a place to experiment.

The portfolio is built around the popular strategy game "Sea Battle".

The portfolio contains:

- [Rules of the game in the Dita format](Rule_of_the_game.dita).<!-- With reuse -->

- [Rules of the game in the reST format](Rule_of_the_game.rst).<!-- Reusing off -->
- [Rules of the game in the AsciiDoc format](Rule_of_the_game.adoc).<!-- With reuse -->

- API reference based on a fictional OpenAPI: [AI generated](API_reference.adoc) (just for fun), [written manually](API_reference.md) (so far only one method has been described).
